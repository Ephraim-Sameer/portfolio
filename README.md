## How to Upload to GitHub

1. Open your GitHub repo:
   `https://github.com/Ephraim-Sameer/portfolio`

2. Upload these files:
   - `index.html`
   - `style.css`
   - `script.js`
   - `README.md`
   - `vercel.json`

3. Make sure your images are also uploaded in the same root folder.

4. Commit the changes.

## Vercel Settings

Use these settings:

- Framework Preset: Other
- Root Directory: `./`
- Build Command: leave empty
- Output Directory: `./`
- Install Command: leave empty

Then click Redeploy.

## Edit Contact Info

In `index.html`, replace:

```html
<a href="mailto:example@email.com">Email Me</a>
```

with the real email address.
